Fundamentos de Sistemas

· Interfaz: Barra de direcciones, elementos visuales como botones, menús y barras propias del navegador.
· Motor interno: Coordina las acciones y comportamientos de otros componentes internos.
· Almacenamiento: Interno, manejo de cookies y ligera base de datos.
· Motor de renderización: Analiza el HTML y CSS para mostrar el contenido en pantalla.
· Red: Componentes HTTP y de otros protocolos de comunicación
· Interprete JS: Analiza y ejecuta el código JS

Formularios WEB

· Formulario web de contacto

URLS

¿Qué es?

· Permite identificar a través de letra, números o símbolos los archivos localizados en la red de internet.
· Se utiliza a través de la barra de busqueda del navegador.
· Cada pagina, cada web y cada recurso disponible en internet tiene una dirección url.

Partes:

· Un esquema: es el protocolo que se utiliza para recuperaar la información.
· Servicio: elemento que cuenta toda url (www)
· Dominio: indica la procedencíia del sitio web
